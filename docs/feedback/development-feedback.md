## 🤝 协作指南

### 给其他 Agent 的建议

1. **开始工作前**:
   - 阅读完整设计文档
   - 检查本反馈文档的最新更新
   - 确认当前 Phase 和任务分配

2. **遇到问题时**:
   - 按照反馈格式模板记录问题
   - 尝试至少 2 种解决方案
   - 记录最终方案和经验教训

3. **提交代码前**:
   - 更新反馈文档
   - 标注相关代码位置
   - 更新进度跟踪表

4. **沟通建议**:
   - 重大决策记录在"设计决策记录"部分
   - 紧急问题标记为 P0 优先级
   - 保持文档同步更新
